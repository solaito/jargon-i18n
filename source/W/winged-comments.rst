.. _winged-comments:

============================================================
winged comments
============================================================

n\.

Comments set on the same line as code, as opposed to :ref:`boxed-comments`\.
In C, for example:

.. code-block:: none


   d = sqrt(x*x + y*y);  /* distance from origin */

Generally these refer only to the action(s) taken on that line.
